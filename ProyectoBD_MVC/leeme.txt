Para probar el funcionamiento de este programa debes:
1. crear una base de datos MySQL llamada 'POO'
2. crear una tabla llamada 'persona' con los siguientes atributos:
   - id: int (PK y A-I)
   - rut: varchar
   - apellidos: varchar
   - nombres: varchar
   - genero: int (1=Fem, 2=Mas)
   - fono: int
3. Falta agregar "validaciones" y en algunos casos control de excepciones
4. Este programa, si lo estimas conveniente, lo puedes usar como modelo para la EVA4
